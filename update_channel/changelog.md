# v24

### Changelog

修复部分手机由系统bug导致的电量无法显示100%的问题。

部分手机满电时的真实电量文件的数值并不是100%，这就导致了程序获取到的真实电量一直卡在99%而实际上手机已经充满电。

因为上述原因，可能会导致充满电时程序所写入的UI电量为100%，而scene中读取到的电量不为100%，此时手机充电电流已经降至0A，手机在物理意义上已经满电，只不过由于系统bug导致真实电量文件的数值不是100%，请不要担心。
